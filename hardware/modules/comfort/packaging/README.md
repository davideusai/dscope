# 📦 D‑Scope Packaging

This folder contains everything related to the physical packaging of D‑Scope modules.

## Contents
- `3d/` → 3D enclosure files
- `purchased/` → References for purchased plastic enclosures
- `labels/` → Labels, QR codes, identifiers
- `notes/` → Miscellaneous packaging information

Each module is delivered in a ready‑to‑use plastic enclosure.
