# .knowledge/

Ce dossier contient toutes les ressources de recherche utilisées pour le développement du standard D‑Scope.  
Il ne fait pas partie de la documentation officielle et n’est pas destiné à être publié dans les releases.

## Structure

- **sources/**  
  PDF, articles techniques, documents de référence, publications.

- **notes/**  
  Notes personnelles, résumés, analyses, réflexions.

- **links/**  
  Collections d’URLs, ressources web, pages importantes.

- **drafts/**  
  Brouillons, idées, schémas, explorations conceptuelles.

- **sketches/**  
  Ce dossier contient des expérimentations techniques, des prototypes rapides et des essais de code liés au développement du standard D‑Scope.

## Objectif

Fournir un espace clair, organisé et durable pour centraliser toutes les ressources de recherche, sans polluer la structure du projet ni la documentation officielle.
